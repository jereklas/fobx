## Why static-docs

This package builds docs as static HTML files with a navigation layout that
behaves like a lightweight single-page app.

## Highlights

- Left navigation with frontmatter overrides
- Right side table of contents
- Client-side search and theme switching
- Static output ready for GitHub Pages
